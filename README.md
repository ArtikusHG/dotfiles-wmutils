# dotfiles-wmutils
My final, most advanced, wmutils dotfiles.
# Requirements
Mandatory: Lemonbar, urxvt, cherry (font), wmutils, sxhkd, feh (or any other wallpaper manager), dunst, compton
Optional: cava, ncmpcpp
# Credits
